# User Stories Registry

Central repository for all user stories across the Receptor Ecosystem.

This registry tracks the implementation status of features for each application in the suite, linking high-level requirements to actual code and technical tasks.

## Application Registries

- **[Preference Frontend](./preference-frontend/index.md)**: Worker-facing application for submitting rotation preferences.
- **[Workforce Frontend](./workforce-frontend/index.md)**: Administrator application for managing allocation runs and hospital data.
- **[Planner Frontend](./planner-frontend/index.md)**: Planner implementation for running matching algorithms and analyzing results.
